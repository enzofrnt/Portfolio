Contexte

Ce mémoire présente une étude approfondie sur l'implémentation d'un système RAG (Retrieval-Augmented Generation) pour l'analyse automatique de rapports. Le projet explore les différentes approches et technologies permettant d'améliorer la compréhension et l'extraction d'informations à partir de documents techniques.
